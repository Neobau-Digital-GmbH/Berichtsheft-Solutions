# Berichtsheft

Eine kleine Mac-App zum Ausfüllen und Drucken des wöchentlichen
Ausbildungsnachweises (Berichtsheft).

## ⬇️ Herunterladen

**[Berichtsheft herunterladen (Version 1.0)](https://raw.githubusercontent.com/Neobau-Digital-GmbH/Berichtsheft-Solutions/main/Berichtsheft-1.0.zip)**

## Installieren

1. Die heruntergeladene Datei per Doppelklick entpacken.
2. **`BerichtsheftApp`** in den Ordner **Programme** ziehen.
3. Beim **allerersten** Start: mit **Rechtsklick** auf die App → **„Öffnen"** →
   nochmal **„Öffnen"**. (Nötig, weil die App nicht über den App Store kommt —
   danach startet sie ganz normal per Doppelklick.)

## Updates

Neue Versionen bekommst du automatisch: Die App meldet sich, sobald ein Update
da ist, und installiert es auf einen Klick. Du kannst auch selbst über das Menü
**Berichtsheft → „Nach Updates suchen…"** prüfen.

Deine Daten (Stammdaten, Wochen, PDFs) liegen sicher in deinem iCloud-Ordner —
ein Update ändert daran nichts.

## Voraussetzungen

macOS 26 oder neuer, Apple-Silicon-Mac (M1 oder neuer).
